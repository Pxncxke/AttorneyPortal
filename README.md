## Setup & Installation


```bash
Clone the repository:
git clone https://github.com/Pxncxke/AttorneyPortal.git

```
```bash
Install dependencies:
npm install
# or
yarn install
```
## Run the development server

```bash
npm run dev
# or
yarn dev
Open http://localhost:3000 to view the site.
```
